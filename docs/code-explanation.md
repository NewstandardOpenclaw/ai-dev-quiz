# AWS DVA クイズアプリ コード解説

## アプリ全体の構造

```
┌─────────────────────────────────────────┐
│           AWS DVA クイズアプリ           │
│                                         │
│  スタート画面 → クイズ画面 → 結果画面   │
│      ↕                                  │
│  統計画面                               │
└─────────────────────────────────────────┘
```

4つの「画面（Screen）」があり、`phase`（フェーズ）という変数で今どの画面を表示するかを切り替えています。

---

## ファイルの役割

| ファイル | 役割 |
|---|---|
| `src/App.tsx` | アプリ全体の司令塔。データ取得・画面切り替えを管理 |
| `src/components/StartScreen.tsx` | スタート画面。フィルター設定・クイズ開始 |
| `src/components/QuizScreen.tsx` | クイズ画面。問題表示・回答処理 |
| `src/components/ResultScreen.tsx` | 結果画面。スコア表示 |
| `src/components/StatsScreen.tsx` | 統計画面。過去の回答履歴を分析 |

---

## 各画面の説明

### 1. スタート画面（StartScreen.tsx）

```
カテゴリ選択: [S3] [Lambda] [DynamoDB] ...
難易度選択:   [easy] [medium] [hard]
出題数選択:   [1問] [5問] [10問] ...
シャッフル:   □ チェックボックス

[既存の問題でスタート]
[統計を見る]

── AIで問題を生成 ──
[S3] [Lambda] [DynamoDB] ...  ← AIが即席で問題を作る
```

**何をしているか：**
- ユーザーが選んだ条件（カテゴリ・難易度・出題数・シャッフル）を `useState` で記憶
- 「スタート」ボタンを押すと、条件を App.tsx に渡す
- 「AIで生成」ボタンは Supabase Edge Functions を呼び出してAIが問題を作る

---

### 2. クイズ画面（QuizScreen.tsx）

```
問題 2 / 10   ───────── 10%

[Lambda] · medium

問題文がここに表示される

○ 選択肢A
○ 選択肢B   ← クリックで選択
○ 選択肢C
○ 選択肢D

（回答後）解説文が表示される

[次へ →]
```

**何をしているか：**
- 選択肢をクリックすると正解は緑、不正解は赤で色が変わる
- `selected` が `null` の間は何度クリックしても変わらない（二度押し防止）
- 「次へ」を押すと `selected` をリセットして次の問題へ

---

### 3. 結果画面（ResultScreen.tsx）

```
クイズ完了
   75%
10問中 7問正解

[もう一度挑戦]  [統計を見る]
```

シンプルに正答率を計算して表示するだけです。

---

### 4. 統計画面（StatsScreen.tsx）

一番複雑な画面です。

```
フィルター: カテゴリ × 難易度

総合正答率: 68%  (50問中34問)

カテゴリ別:
  Lambda  ████████░░ 80%  (8/10)
  S3      ██████░░░░ 60%  (6/10)

難易度別:
  easy    ████████░░ 80%
  hard    ████░░░░░░ 40%

DVA試験ドメイン別:
  開発 (32%)          正答率 ██████ 70%
  セキュリティ (26%)  正答率 ████░░ 50%

問題別（正答率の低い順）:
  [S3 · hard] 20%  S3のライフサイクル設定で...
```

**何をしているか：**
- Supabase の `quiz_results` テーブルから全回答履歴を取得
- フィルター変更時は `useMemo` で再計算（DBは叩かず、手元のデータを絞り込む）
- DVA試験の4ドメイン別に正答率を表示（本番試験の出題比率と比較できる）

---

## データの流れ（全体像）

```
Supabase DB
  │
  ├─ quizzes テーブル（問題データ）
  │     ↓ App.tsx が起動時に全件取得
  │     ↓ スタート時にフィルター＋件数制限
  │     ↓ QuizScreen に1問ずつ渡す
  │
  └─ quiz_results テーブル（回答履歴）
        ↑ 最後の問題に答えたとき一括保存
        ↓ StatsScreen が全件取得して分析
```

---

## まとめ

このアプリは「AWS Developer Associate 試験対策クイズ」で、主な機能は：

1. **既存問題を解く** — カテゴリ・難易度・出題数を絞って練習
2. **AIで問題を生成** — Supabase + AI を使って即席で問題作成
3. **統計で弱点確認** — 過去の回答を分析して苦手なカテゴリを把握
